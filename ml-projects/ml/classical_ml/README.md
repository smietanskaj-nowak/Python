# Classical Machine Learning

This directory contains projects based on **classical machine learning methods**,
where model structure, features, and assumptions are explicitly defined.

The focus is on **interpretable models, feature engineering, and evaluation**
for structured and semi-structured data.

### Topics include:
- regression and classification
- decision trees and ensemble methods
- feature importance and model explainability
- cross-validation and model comparison

Technologies: scikit-learn, XGBoost, H2O, pandas, NumPy.
