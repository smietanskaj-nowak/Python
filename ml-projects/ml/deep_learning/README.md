# Deep Learning

This directory contains projects based on **neural network architectures**
implemented using modern deep learning frameworks.

The focus is on **model architecture, training dynamics, and performance analysis**
for unstructured data such as images and text.

### Topics include:
- feed-forward neural networks
- convolutional neural networks (CNNs)
- training and evaluation workflows
- performance and scalability considerations

Technologies: PyTorch, TensorFlow, Keras.
